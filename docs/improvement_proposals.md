2. **ズームUIボタン追加**
   - 位置: ツールバー右側

   - 効果: マウス操作が苦手な受講者への配慮

3. **ズーム倍率表示**
   - 位置: 画面右下
   - 表示: `125%` `100%` など
   - 効果: 現在の倍率を可視化

**実装工数**: 3〜5時間  
**優先度**: ★★★★★（高）  
**期待効果**: UI満足度 4.33 → 4.8 への向上
