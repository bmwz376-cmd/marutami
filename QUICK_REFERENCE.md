## 🎮 操作早見表

### 講師モード

| 操作 | 方法 |
|------|------|
| ページ送り | `←` `→` キー or ボタン |
| ズーム | マウスホイール or `+` `-` |
| パン | ドラッグ |
| ピン追加 | 「ピン」選択 → クリック |
| レーザー | 「レーザー」選択 → クリック（3秒で消去）|
| 重要ポイント | 右パネル → カード選択 |
| 同期ON/OFF | 右パネル「制御」タブ |

### 受講者モード

| 操作 | 方法 |
|------|------|
| 同期中 | ページ送り不可、ズーム・パンは可 |
| 同期OFF時 | 自由にページ送り可 |
| 用語確認 | 右パネル「用語」タブ |
| チェックリスト | 右パネル「チェックリスト」タブ |
| フィードバック | 右下「フィードバック」ボタン |
